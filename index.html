A/

What lands in the package

CUT LIST

Validated BOM

Every part with finished L×W×thickness, grain, and banding. Sheet-fit checked, edge-banding totalled. Export md, csv, json, or xlsx.

DRAWINGS

Dimensioned sheets

Plan, front and side elevations with real dimension chains — plus a print-clean packet PDF with a title block.

JOINERY

Drilling coordinates

Per-part hole positions from verified Häfele/Blum data — never guessed. Cam-and-dowel, confirmat, dowels, System-32.

PREVIEW

Interactive 3D

A self-contained three.js render for sign-off — real materials, soft shadows, and open/close toggles for moving parts.

ASSEMBLY

Build order

A reachability-sorted, step-by-step plan with the hardware list and tools — out-of-envelope joints flagged, never faked.

MODEL

SketchUp .skp

With the SketchUp MCP connected, a real editable model built from the same spec — mm converted only at the boundary.

B/

Add it to your Claude

Get the files

Grab the installable skill, or the whole repo if you want the source and examples too.

Download .skill Download repo .zip

Unzip it into your skills folder

Both files unzip to a furniture-design/ folder. Put that folder where Claude looks for skills.

Windows
%USERPROFILE%\.claude\skills\furniture-design
macOS / Linux
~/.claude/skills/furniture-design

Restart Claude and start a new chat

Claude Code and Claude Desktop discover skills on startup. Confirm the folder holds SKILL.md at its top level, then reload.

# Claude Code — verify it's discovered
claude   # then ask: "design me a 900mm oak sideboard"

Optional — connect SketchUp

For the editable .skp, enable the Trimble SketchUp MCP connector. The skill drives it as the 3D backend; everything else works without it. See sketchup-integration.md.

C/

The rule that keeps material out of the scrap pile

Never derive a real dimension from a photo. Images set style and intent; numbers come from a tape measure or a standard. The skill's whole job is to hold, compute, cross-check, and lay out those numbers — one millimetre spec, every view and the cut list derived from it, nothing hand-patched. Assumed numbers stay flagged until you confirm them.
